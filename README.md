# letscook
રોજ "જમવામાં શું બનાવવું?" એ ભૂલી જાવ! આરોગ્ય નિધિ તમારી મનપસંદ ગુજરાતી અને હેલ્ધી વાનગીઓનું 28-દિવસનું ઓટોમેટિક, નોન-રિપીટીંગ શેડ્યૂલ બનાવે છે.  સ્માર્ટ રોટેશન: સવારનો નાસ્તો અને રાતનું ભોજન.  પ્રીપેરેશન એલર્ટ્સ: કઠોળ પલાળવા માટે રિમાઇન્ડર.  ગ્રોસરી લિસ્ટ અને રેસીપી વિડિઓઝ.  તમારું કુકિંગ ટેન્શન-ફ્રી બનાવો!
